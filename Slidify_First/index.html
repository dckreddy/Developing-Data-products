Introduction to Uniform Distribution App

  1. The app is built using shiny and hosted on shinyapps.io.

  2. You can try it here.

  3. This app generates probability density function for the given input.

Math Formulae

  1. Uniform distribution random variable is the form \(U\ (a,b)\)

  2. Probability Distribution Function is $\(f(x)=\frac{1}{b-a} for a <=x<= b;0 for x<a or x>b\)$

R code

  1. The probability is calculated using R function punif() 

u <- runif(10)
punif(10)

  1. Input the value of x to find the Probability Density Function.
